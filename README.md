# AVM-Ajex

Ajex 相当于 AVM 的调度器。它将大型合约拆分为多个执行过程，执行完一个过程即切换至其他合约，避免单合约长期阻塞系统。

**费用模型：**

| 费用类型 | 计算依据 |
|---------|---------|
| 链下执行费用 | RISC-V 指令数量 |
| 链上资源使用费用 | 具体项目单价（测试网确认） |
| 验证费用 | 证明长度与复杂度 |
| 上下文切换费用 | Ajex 过程切换次数，次数越多越贵 |
| 优先级费用 | 优先级系数 n，费用 = 原价^n |
| 繁忙系数 | 网络负载，越繁忙越贵 |

此外，Ajex 提供原生异步事件系统（合约可注册时间或条件触发任务，无需外部 Keeper 服务）、合约中断与恢复、通过瞬时地址映射和三段数据加密实现的隐私执行。
